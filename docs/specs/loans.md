# 貸出機能 仕様書

## 概要

書籍の貸出・返却を管理する機能。1冊の書籍は同時に1人の会員にのみ貸し出せる。

共通のページネーション・エラー形式・ステータスコードは [common.md](./common.md) を参照。

## 用語・ルール

- **貸出期間**: 14日間固定
- **貸出中**: `loans.returned_at` が `null` の状態
- **延滞**: `due_date` が現在日付より前で、かつ貸出中の状態

## 貸出開始

```
POST /loans
```

**リクエストボディ**

```json
{ "bookId": 1, "memberId": 1 }
```

**処理ステップ**

1. リクエストボディを受け取る
2. `bookId` / `memberId` を必須項目として検証する。不正な場合は `400 Bad Request`（[共通エラー形式](./common.md#エラーレスポンス形式)）
3. `bookId` / `memberId` に対して[参照先IDの存在チェック](./common.md#参照先idの存在チェック)を行う
4. 指定した `bookId` の書籍が貸出中（`returned_at` が `null` のレコードが存在）でないか確認する。貸出中の場合は `409 Conflict`
5. `loaned_at` に現在日時、`due_date` に現在日付+14日を設定してレコードを `INSERT` する
6. `Location: /loans/{id}` ヘッダーを付与して `201 Created` を返す

## 返却

```
PATCH /loans/{id}/return
```

**処理ステップ**

1. `id` に一致する貸出レコードが存在するか確認する。存在しない場合は `404 Not Found`
2. 既に返却済み（`returned_at` が非 `null`）でないか確認する。返却済みの場合は `409 Conflict`
3. `returned_at` に現在日時を設定して `UPDATE` する
4. `200 OK` を返す

## 貸出一覧取得

```
GET /loans
```

**クエリパラメータ**

| 名前 | 型 | 必須 | 説明 |
|---|---|---|---|
| memberId | long | 任意 | 会員で絞り込み |
| bookId | long | 任意 | 書籍で絞り込み |
| page, size | - | 任意 | [共通仕様](./common.md#ページネーション)参照 |

**処理ステップ**

1. クエリパラメータを受け取る
2. `page` / `size` 未指定時は[共通仕様](./common.md#ページネーション)のデフォルト値を適用
3. `memberId` / `bookId` が指定されている場合、該当条件で絞り込む
4. 貸出日時の降順で取得しページング処理を行う
5. `200 OK` で返却する

## 延滞一覧取得

```
GET /loans/overdue
```

**処理ステップ**

1. クエリパラメータ（`page` / `size`）を受け取り、[共通仕様](./common.md#ページネーション)のデフォルト値を適用
2. `returned_at` が `null` かつ `due_date` が現在日付より前のレコードを抽出する
3. `due_date` の昇順（延滞が長いものが先頭）で取得しページング処理を行う
4. `200 OK` で返却する
