# 著者機能 仕様書

## 概要

著者情報の登録・参照・更新・削除を行う機能。書籍とは多対多で関連付けられる。

共通のページネーション・エラー形式・ステータスコードは [common.md](./common.md) を参照。

## 一覧取得

```
GET /authors
```

**クエリパラメータ**

| 名前 | 型 | 必須 | 説明 |
|---|---|---|---|
| page, size | - | 任意 | [共通仕様](./common.md#ページネーション)参照 |

**処理ステップ**

1. クエリパラメータを受け取る
2. `page` / `size` 未指定時は[共通仕様](./common.md#ページネーション)のデフォルト値を適用
3. 著者を名前順に取得しページング処理を行う
4. `200 OK` で返却する

## 詳細取得

```
GET /authors/{id}
```

**処理ステップ**

1. `id` に一致する著者を取得する
2. 存在しない場合は `404 Not Found` を返す
3. 存在する場合、`200 OK` で返却する

## 登録

```
POST /authors
```

**リクエストボディ**

```json
{ "name": "夏目漱石" }
```

**処理ステップ**

1. リクエストボディを受け取る
2. `name` を検証する（必須、1〜255文字）。不正な場合は `400 Bad Request`（[共通エラー形式](./common.md#エラーレスポンス形式)）
3. 著者レコードを `INSERT` する
4. `Location: /authors/{id}` ヘッダーを付与して `201 Created` を返す

## 更新

```
PUT /authors/{id}
```

**処理ステップ**

1. `id` に一致する著者が存在するか確認する。存在しない場合は `404 Not Found`
2. `name` を登録時と同じルールで検証する。不正な場合は `400 Bad Request`
3. 著者レコードを `UPDATE` する
4. `200 OK` を返す

## 削除

```
DELETE /authors/{id}
```

**処理ステップ**

1. `id` に一致する著者が存在するか確認する。存在しない場合は `404 Not Found`
2. `book_authors` に当該著者を参照するレコードがないか確認する。存在する場合は `409 Conflict`（[共通処理](./common.md#参照整合性による削除制限)）
3. 著者レコードを削除する
4. `204 No Content` を返す
