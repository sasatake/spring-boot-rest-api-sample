# カテゴリ機能 仕様書

## 概要

書籍を分類するカテゴリの登録・参照・削除を行う機能。書籍とは多対多で関連付けられる。
更新頻度が低いため更新（PUT）は提供しない。

共通のエラー形式・ステータスコードは [common.md](./common.md) を参照。

## 一覧取得

```
GET /categories
```

**処理ステップ**

1. カテゴリを名前順に全件取得する（件数が少なく想定されるためページングは行わない）
2. `200 OK` で配列を返却する

## 登録

```
POST /categories
```

**リクエストボディ**

```json
{ "name": "小説" }
```

**処理ステップ**

1. リクエストボディを受け取る
2. `name` を検証する（必須、1〜255文字）。不正な場合は `400 Bad Request`（[共通エラー形式](./common.md#エラーレスポンス形式)）
3. カテゴリレコードを `INSERT` する
4. `Location: /categories/{id}` ヘッダーを付与して `201 Created` を返す

## 削除

```
DELETE /categories/{id}
```

**処理ステップ**

1. `id` に一致するカテゴリが存在するか確認する。存在しない場合は `404 Not Found`
2. `book_categories` に当該カテゴリを参照するレコードがないか確認する。存在する場合は `409 Conflict`（[共通処理](./common.md#参照整合性による削除制限)）
3. カテゴリレコードを削除する
4. `204 No Content` を返す
